# nowt36

以下の特徴を持ったキーボードです。

* 自然と手が置ける配列
* 一体型だけど分割型
* 片手5列
* ホットスワップ対応
* ミドルアクリルプレート対応 (オプション)
* アンダーグロー対応 (オプション)
